# Service Certify: Updated Product Requirements Document (Exam Focus) ## 1. Goal Create a specialized practice exam platform for ServiceNow certifications. The site provides high-quality mock exams and domain-specific practice tests to help technical architects and developers prepare for official ServiceNow certifications. It operates on a subscription model for full access to all test banks. ## 2. Core User Flows 1. **Discovery:** User explores mock exams for specific ServiceNow certifications (e.g., CSA, CAD, CIS-ITSM). 2. **Subscription:** User joins the membership for unlimited access to all practice exams. 3. **Practice:** User takes timed mock exams or focused quizzes by product domain. 4. **Performance Tracking:** User monitors their test scores, identifies weak areas, and tracks readiness for the actual exam via a personal dashboard. ## 3. Revised Screens 1. **Home/Landing Page:** Focus on "Exam Readiness," showing pass rates, test categories, and the "Join Membership" CTA. Remove video-related imagery. 2. **Exam Catalog:** A filtered list of ServiceNow certification mock exams and domain practice sets. 3. **Exam Details Page:** Breakdown of what's in the test bank (number of questions, domains covered, last update date) instead of a video curriculum. 4. **Candidate Dashboard:** Focus on "Mock Exam Performance," average scores, and "Readiness Score" for official exams. 5. **Practice Exam Interface:** (Existing {{DATA:SCREEN:SCREEN_2}} is already well-aligned but can be refined if needed). ## 4.
